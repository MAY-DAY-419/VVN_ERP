# Media Folder

Upload your school photos here to display them on the website.

## Instructions:
1. Place all school photos (events, activities, campus, etc.) in this folder
2. Supported formats: JPG, JPEG, PNG
3. Recommended image size: 800x600px or larger
4. Name your files descriptively (e.g., annual-day-2026.jpg, sports-day.jpg)

## Examples:
- annual-day-2026.jpg
- sports-event.jpg
- campus-view.jpg
- classroom.jpg
- students-activities.jpg
